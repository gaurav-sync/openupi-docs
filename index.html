📱 openUPI - Professional UPI QR Scanner

A fast, professional UPI QR code scanner for Android with native app integration

Features • Screenshots • Installation • Building • Usage • Contributing

✨ Features

🎯 Core Functionality

  • Instant Camera Launch - Opens camera immediately on app start
  • Real-time QR Scanning - Continuous scanning with ZXing library
  • UPI Detection - Automatically detects UPI payment links (upi://pay)
  • Native App Chooser - Seamlessly integrates with Android's app selection
  • Gallery Scanning - Scan QR codes from saved images
  • Multi-format Support - Handles various UPI QR code formats

🎨 Professional UI

  • Modern Design - Clean, intuitive interface
  • Blue Corner Brackets - Professional scanning frame
  • Top Toolbar - Gallery access and flash controls
  • Bottom Controls - Zoom slider and quick zoom buttons
  • Visual Feedback - Clear status messages and animations

📷 Camera Controls

  • Flash Toggle - Turn camera flash on/off
  • Zoom Controls - Slider and buttons for precise zoom
  • Auto-focus - Continuous focus for sharp QR detection
  • Camera Resume - Automatic camera restart after UPI transactions

🔒 Privacy & Permissions

  • Minimal Permissions - Only camera and storage (for gallery)
  • No Data Collection - Completely offline operation
  • Secure Processing - QR codes processed locally

📥 Installation

Option 1: Download APK

  1. Download the latest APK from Releases
  2. Enable "Install from unknown sources" in Android settings
  3. Install the APK file

Option 2: Build from Source

git clone https://github.com/yourusername/openUPI.git
cd openUPI
./gradlew assembleDebug

🛠 Building

Prerequisites

  • Android Studio 4.0+ or Android SDK
  • Java 8+ or OpenJDK
  • Gradle 8.0+ (included via wrapper)
  • Android SDK API 24+ (Android 7.0)

Build Commands

# Debug build (for testing)
./gradlew assembleDebug

# Release build (for distribution)
./gradlew assembleRelease

# Clean build
./gradlew clean assembleDebug

# Install on connected device
./gradlew installDebug

Build Outputs

  • Debug APK: app/build/outputs/apk/debug/app-debug.apk
  • Release APK: app/build/outputs/apk/release/app-release.apk

🚀 Usage

Basic Operation

  1. Launch App → Camera starts immediately
  2. Point at QR Code → Blue brackets guide alignment
  3. UPI Detected → Native Android app chooser appears
  4. Select UPI App → Complete payment in chosen app
  5. Return to Scanner → Ready for next scan

Advanced Features

  • Gallery Scanning: Tap gallery icon to scan saved QR images
  • Flash Control: Toggle flash for low-light scanning
  • Zoom Control: Use slider or buttons for precise zoom
  • Continuous Scanning: Automatically resumes after each scan

Supported UPI Formats

upi://pay?pa=merchant@bank&pn=Name&am=100

QR codes with UPI parameters (pa=, pn=, mc=)
Standard UPI payment links from all major providers
